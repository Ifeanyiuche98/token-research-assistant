Token Research Assitant.

🚀 Live Demo: https://token-research-assistant.vercel.app/

A lightweight web app that generates quick research notes for crypto tokens and Web3 projects.

Users can enter a token or project name and instantly receive:

- Summary
- Use case
- Key risks
- Ecosystem notes

Built with:
- React
- TypeScript
- Vite

Deployed on Vercel.

## 🚀 Token Comparison Engine

The Token Research Assistant now includes a side-by-side token comparison engine.

Users can compare two crypto tokens or projects instantly using live market data.

### Features
- Side-by-side token comparison
- Live market metrics (CoinGecko)
- Automatic fallback dataset
- Highlight stronger metrics
- Swap tokens instantly
- Clean research summary view

### Example Comparison Fields
- Price
- 24h Change
- Market Cap
- 24h Volume
- Fully Diluted Valuation
- Project summary
- Categories
- Official links

This feature allows researchers and investors to quickly evaluate the relative strength of two crypto projects.
