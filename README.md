Token Research Assitant.

🚀 Live Demo: https://token-research-assistant.vercel.app/

A lightweight web app that generates quick research notes for crypto tokens and Web3 projects.

Users can enter a token or project name and instantly receive:

- Summary
- Use case
- Key risks
- Ecosystem notes

Built with:
- React
- TypeScript
- Vite

Deployed on Vercel.
